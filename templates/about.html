About this Project:

This project was intended to solely be a fun personal project and therefore lacks a lot of important elements that make up a proper e-commerce website. The bulk of the project was implementing the search bar in the main page which took up much more time than I anticipated. It required cleaning the original data and also implementing Natural Language Processing techniques to filter through the dataset and find the most similar products based on the user's search.

Given the time for improvements:

- I would make the website much more aesthetic

- I would create a more robust machine learning algorithm given more data by clustering similar users together and creating a recommendation algorithm for similar users

- I would add more webpages to make the website more functional
